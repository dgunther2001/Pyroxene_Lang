Start Guide

Clone the repository

  • git clone https://github.com/dgunther2001/pyroxene_lang

Install and enable Docker and/or Docker Desktop

  • GUI: Docker
  • CLI:
      - On Ubuntu CLI:
        - sudo apt-get install -y apt-transport-https ca-certificates curl software-properties-common
        - curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg
        - echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
        - sudo apt-get update
        - sudo apt-get install -y docker-ce docker-ce-cli containerd.io
        - sudo systemctl start docker
        - sudo systemctl enable docker

Execution

Zsh and Bash Shells

  • Running Language Without Aliases:
      - Debug Mode:
        - chmod u+x run_debug_driver.sh
        - ./run_debug_driver.sh /path/to/your/pyroxene/file
      - Regular Mode:
        - chmod u+x run_driver.sh
        - ./run_driver.sh /path/to/your/pyroxene/file
  • Running Language With Aliases:
      - chmod u+x run_debug_driver.sh run_driver.sh setup.sh
      - ./setup.sh
      - Bash: source ~/.bashrc
      - Zsh: source ~/.zshrc
      - Debug Mode:
        pyroxene /path/to/your/pyroxene/file
      - Regular Mode:
        pyroxene-debug /path/to/your/pyroxene/file
