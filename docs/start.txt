To toggle on and off debug mode: 

=> go into project root directory
=> for debug mode, type:
    ./run_debug_driver.sh /path/to/your/pyroxene/file
=> for the regular language driver, type:
    ./run_driver.sh /path/to/your/pyroxene/file
